# Anniversary Story Website

A fully responsive luxury Netflix-inspired anniversary website built with React, Vite, Tailwind CSS, Framer Motion, and React Router.

## Project Features

- Cinematic landing experience with animated storytelling
- Four seasons of episodes representing 21 years of family memories
- Expandable season rows and interactive episode cards
- Episode detail pages with memory galleries and emotional quotes
- Glassmorphism, luxury red/beige theme, and film grain overlays
- Background audio control with soft music toggle
- Tailwind CSS integration and reusable component architecture

## Local Development

Install dependencies and start the site locally:

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Notes

- The design is built to feel cinematic, premium, and nostalgic.
- Episode galleries load from `/public/memories/[year]` with placeholder fallbacks.
- The final episode includes the closing anniversary message for 2026.
